Trophy client

The trophy client is used to fetch the trophies (all, achieven, non achieved) and to achieve trophies.

Examples

Fetch Trophies

private async void FetchSomeTrophies() { TrophyClient trophyClient = new TrophyClient(); // You can fetch all trophies.. IEnumerable allTrophies = await trophyClient.FetchAllTrophies(); // .. or only achieved trophies.. IEnumerable achievedTrophies = await trophyClient.FetchAchievedTrophies(); // .. or only non achieved tophies. IEnumerable nonAchievedTrophies = await trophyClient.FetchNonAchievedTrophies(); // Also you can fetch one or more trophies by their id. IEnumerable selectedTrophies = await trophyClient.FetchTrophyById(5947, 2584); }

Achieve Trophies

To achieve a trophy you only need following line.

await trophyClient.AchieveTrophy(5947);
