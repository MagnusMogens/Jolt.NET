Session Manager

The session manager is used to handle the sessions to gamejolt from one or multiple users.

Examples

In the following example a new instance of the game client will be created. After an user is authentificated a session will be established and automatically pinged.

using Jolt.NET; using Jolt.NET.Network; class Program { SessionManager manager; GameClient client; void Main(string[] args) { client = new GameClient("gameId", "gameKey"); manager = new SessionManager(); client.AuthenticationCompleted += OnAuthenticationCompleted; // The first user who is authenticated is set as default user in the Settings class. client.AuthenticateUser("username", "token"); } private void OnAuthenticationCompleted(object sender, ResponseEventArgs e) { // All functions with an user parameter can be used with or without user. If no // user is specified the current authenticated default User is used. manager.OpenSession(); // By default the user session will now be pinged every 30 seconds with the status 'active'. manager.AutoPingUser(); } }

Ping multiple user

Jolt.NET offers to ping multiple users each with their own session state.

manager.AutoPingUser(30, SessionStatus.Active, Settings.Instance.AuthenticatedUsers[0]); manager.AutoPingUser(30, SessionStatus.Active, Settings.Instance.AuthenticatedUsers[1]); manager.AutoPingUser(30, SessionStatus.Idle, Settings.Instance.AuthenticatedUsers[2]);
